# MVP

## Goal

Build a delightful single-player prototype that proves the core loop is fun.

## MVP features

- day-based progression
- 3 districts
- 5 agents with distinct traits
- 8 mission types
- dynamic events
- cash and reputation systems
- simple inventory categories
- mission composer UI
- end-of-mission reports
- end-of-day summary screen

## Non-goals for MVP

- multiplayer
- full blockchain dependency
- deep contract systems
- large-scale production economy
- real-time simulation

## Best first demo story

The player launches a small bazaar brand, sends out a few agents, makes a few risky decisions, and watches their business become either promising, ridiculous, or both.

## Stretch goals

- district map
- animated agent report feed
- rival brands
- live rumor tape
- premium paid actions backed by Stellar testnet
