# UI Direction

## Tone

Funny management dashboard.
Polished layout. Unserious copy. Serious enough data presentation.

## Visual references

The UI should feel like:
- a luxury market terminal
- a theatrical operations room
- a startup dashboard run by smugglers

## Core screens

- Home command desk
- Agent roster
- Mission planner
- District map
- Merchant and rumor panels
- Incident inbox
- Reports archive

## UI ingredients

- rich cards
- little stamps and receipt motifs
- sliders for budget and risk
- faction badges
- exaggerated report headers
- lovely charts next to ridiculous event copy

## Interaction style

- fast and readable
- rich hover states
- panel-based
- mission reports should feel rewarding to open
- incidents should interrupt politely but dramatically
