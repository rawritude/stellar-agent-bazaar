# Game Loop

## Daily loop

1. Morning brief
- review cash, inventory, trends, district alerts, and unresolved incidents

2. Planning
- choose pricing
- choose what products to push
- allocate budgets
- assign agents to missions
- choose mission posture: cautious, balanced, reckless, theatrical

3. Resolution
- agents perform actions in districts
- merchants react
- hazards and opportunities trigger
- budgets are consumed or preserved
- some actions require payment to unlock deeper opportunities

4. Reports
- each agent returns with a readable mission summary
- each report shows money spent, money earned, rumors found, and side effects

5. Reaction
- player adjusts policy, rewards agents, reshuffles strategy, or launches a follow-up mission

6. End of day
- reputation shifts
- inventory updates
- district conditions change
- new incidents or offers appear

## Moment-to-moment fun

The fun should come from:
- deciding which imperfect agent to trust
- balancing budget against ambition
- reading vivid mission outcomes
- spotting emerging opportunities
- recovering from absurd setbacks

## Important system rule

Every mission should produce:
- financial outcome
- information outcome
- narrative outcome

Even a failed mission should return something useful, embarrassing, or funny.
